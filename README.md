# Budget de Léa - Brésil Automne 2025 🇧🇷

Application de gestion de budget pour l'échange au Brésil. Interface simple pour suivre vos finances en CAD et BRL.

## ✨ Fonctionnalités

- **📊 Tableau de bord** - Vue d'ensemble de vos finances avec graphiques
- **📅 Suivi mensuel** - Navigation par mois avec comparaison budget vs réalité  
- **💱 Bi-devise** - Support CAD/BRL avec taux de change configurable
- **⚙️ Configuration** - Gestion des dates, soldes, et transactions récurrentes
- **📱 PWA** - Installation sur mobile, fonctionne hors-ligne
- **💾 Sauvegarde** - Export/import de vos données

## 🚀 Utilisation

Ouvrez `index.html` dans votre navigateur ou installez l'app sur votre mobile via le bouton d'installation.
